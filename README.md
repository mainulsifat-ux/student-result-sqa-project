# Student Result Management System (SQA Project)

## Project Overview
This project demonstrates Software Quality Assurance (SQA) techniques by implementing a simple Student Result Management System with automated testing.

## Features
- Student result calculation
- Grade evaluation
- Input validation
- Automated test cases

## Technologies Used
- Python
- PyTest
- GitHub

## How to Run the Project

Run the main program:

python result_system.py

## Run Automated Tests

pytest

## Example Output

Enter student name: Rahim
Enter marks: 85

- Unit Testing
- Input Validation
- Automated Testing
